Insertion Sort
===

## Definition
陣列起頭從 0 開始，若左邊元素較大，則與之互換。看起來就像插入一樣。不斷地插入新的值至排序好的陣列。

## Performance
* Worst Case: O(n^2)
* Best Case: O(n)
* Average Case: O(n^2)

## Code:
```
function insertionSort(arr) {
  for (let i = 1; i < arr.length; i++) {
    const value = arr[i];
    let j = i - 1;
    while (j >= 0 && arr[j] > value) {
      arr[j + 1] = arr[j];
      j--;
    }
    arr[j + 1] = value;
  }
  return arr;
}
```
